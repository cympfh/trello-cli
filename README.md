# a trello CLI

## Requirements

- Bash (sh)
- curl
- [jq](https://stedolan.github.io/jq/download/)

# Install

Add path to the `bin/`

```bash
# bashrc, zshrc
export PATH=/path/to/trello-cli/bin:$PATH
```

## for zsh user

```bash
ln -s /path/to/trello-cli/zsh_completion/_trello ~/.zsh/completion/
```
